# Work in progress.
## A DAW in the browser, with VST support (using VstWeb).
Done:
- MIDI editor
- VST config
- VST host (with some basic features, like loading plugins, saving/loading state, and audio processing)
- Basic UI for the above features, with a simple window management system.

Clone:
```bash
git clone https://github.com/4rchXceed/WebDAW.git
cd WebDAW
git submodule update --init --recursive
```
